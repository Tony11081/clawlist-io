Security & Permissions

OAuth Scope

This skill requires repo:status and public_repo permissions to function correctly. All tokens are stored in hardware-encrypted vaults.

Data Privacy

OpenClaw does not store issue content. All data is processed in-memory and discarded after the session ends. No telemetry is collected from private repositories.
